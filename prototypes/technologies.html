Hypertext Markup Language

Hypertext Markup Language, a standardized system for tagging text files
to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
Example:
html head title title head body body html
It's very important for most websites run on HTML files.

Cascading Style Sheets

Cascading Style Sheets, is a stylesheet language used to describe the presentation of a
document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML).
Example:
div {
opacity: 0.3;
filter: alpha(opacity=30);
}
Also useful for Web designing!

MySQL

MySQL, is an Oracle-backed open source relational database management
system (RDBMS) based on Structured Query Language (SQL).
Example:

SELECT m.dept_no, m.emp_no, d.dept_name FROM dept_manager_dup m INNER JOIN departments_dup d ON m.dept_no = d.dept_no ORDER BY m.dept_no ASC;
The application is used for a wide range of purposes, including data warehousing, e-commerce, and logging applications.
